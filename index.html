Learn 2 Install

Java

  1. Install the JDK: Download "Java Platform (JDK)" from Oracle OTN. Run the installer, using all the defaults.

NetBeans IDE

  1. Download "NetBeans IDE" installer from http://netbeans.org/downloads/index.html.

Eclipse IDE

  1. Download Eclipse from https://www.eclipse.org/downloads. Under "Get Eclipse Neon" ⇒ Click "Download Packages". For beginners, choose the 3rd entry "Eclipse IDE for Java Developers" (32-bit or 64-bit) (e.g., "eclipse-java-neon-2-win32-x86_64.zip" 161MB) ⇒ Download.

Oracle JDeveloper IDE

  1. Go to http://www.oracle.com/technetwork/developer-tools/jdev/downloads

Spring

  1. Go to https://start.spring.io

ColdFusion

  1. Open up a web browser and go to: http://www.adobe.com/go/trycoldfusion
  2. Sign in with your Adobe ID. If you do not have an Adobe account, click on the "Create an Adobe account" button on the left.

SQL Server

  1. Go to http://bit.ly/dl-sql-express-2012

R

R-Project

  1. Go to https://cran.r-project.org/mirrors.html and scroll down to USA and click the first link you can find Usually University of California, Berkeley, CA
  2. Click to download whatever OS version you are running.
      Windows
    1. Click base
    2. Click Download R 3.5.1 for Windows
    3. Next, click through to set all as default values
    4. Click Finish

RStudio

  1. Go to https://www.rstudio.com/products/rstudio/download
  2. Click on Free Download link
  3. Under Installers, click on whatever OS you are running on
      Windows
    1. Click the latest version of RStudio installer
    2. Click through to set all as default values
    3. Click Finish
