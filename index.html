Learn 2 Install

Java

  1. Install the JDK: Download "Java Platform (JDK)" from Oracle OTN. Run the installer, using all the defaults.

NetBeans IDE

  1. Download "NetBeans IDE" installer from http://netbeans.org/downloads/index.html.

Eclipse IDE

  1. Download Eclipse from https://www.eclipse.org/downloads. Under "Get Eclipse Neon" ⇒ Click "Download Packages". For beginners, choose the 3rd entry "Eclipse IDE for Java Developers" (32-bit or 64-bit) (e.g., "eclipse-java-neon-2-win32-x86_64.zip" 161MB) ⇒ Download.

Oracle JDeveloper IDE

  1. Go to http://www.oracle.com/technetwork/developer-tools/jdev/downloads
